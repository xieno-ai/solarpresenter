  <files>src/app/api/extract-bill/route.ts</files>
  <action>
    Create src/app/api/extract-bill/route.ts:

    Imports:
    - NextRequest from 'next/server'
    - GoogleGenAI from '@google/genai'
    - zodToJsonSchema from 'zod-to-json-schema'
    - extractedBillSchema, ExtractBillResult from '@/lib/bill/schema'
    - BILL_EXTRACTION_PROMPT from '@/lib/bill/prompt'

    Runtime config:
    ```typescript
    export const runtime = 'nodejs';
    export const maxDuration = 60;
    ```

    Initialize Gemini client at module scope:
    ```typescript
    const ai = new GoogleGenAI({ apiKey: process.env.GOOGLE_GENERATIVE_AI_API_KEY! });
    const GEMINI_MODEL = process.env.GEMINI_MODEL ?? 'gemini-2.5-flash';
    ```

    POST handler logic:
    1. Parse request.formData() — wrap in try/catch, return 400 on failure
    2. Get file = formData.get('file') as File | null — return 400 if missing
    3. Validate MIME type: allowed = ['application/pdf', 'image/jpeg', 'image/png', 'image/heic', 'image/heif'] — return 400 with "Unsupported file type" if not in list
    4. Validate file size: file.size > 4 * 1024 * 1024 → return 413 with "File exceeds 4MB limit"
    5. Convert to base64: Buffer.from(await file.arrayBuffer()).toString('base64')
    6. Call Gemini:
       ```typescript
       const response = await ai.models.generateContent({
         model: GEMINI_MODEL,
         contents: [
           { text: BILL_EXTRACTION_PROMPT },
           { inlineData: { mimeType: file.type, data: base64Data } },
         ],
         config: {
           responseMimeType: 'application/json',
           responseSchema: zodToJsonSchema(extractedBillSchema),
         },
       });
       ```
    7. Parse and validate response:
       ```typescript
       const raw = JSON.parse(response.text ?? '{}');
       const fields = extractedBillSchema.parse(raw);
       ```
    8. Determine status: if all fields are null → 'error' with message "Could not extract data from this document"; if some fields null → 'partial'; else → 'success'
    9. Return Response.json({ status, fields }) — always HTTP 200 (same convention as /api/scrape)
    10. Wrap the Gemini call in try/catch: on error return Response.json({ status: 'error', fields: null, message: error.message })

    Do NOT set Content-Type header on the fetch — browser sets multipart boundary automatically (client responsibility, but document this in a comment).
