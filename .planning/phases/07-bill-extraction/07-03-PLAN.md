    Loading interval (mirror SunPitchTab exactly):
    - useEffect watching `phase === 'loading'`: setStatusIndex(0), start setInterval 2500ms cycling
    - Cleanup clears interval on phase change or unmount

    handleUpload(file: File):
    1. Client-side validation: file.size > 4MB → setUploadError('File too large — please use a file under 4 MB.'); return
    2. MIME check: if file.type not in ['application/pdf', 'image/jpeg', 'image/png', 'image/heic', 'image/heif'] → setUploadError('Unsupported file type. Please upload a PDF, JPG, PNG, or HEIC file.'); return
    3. setPhase('loading'), setUploadError(null)
    4. const formData = new FormData(); formData.append('file', file);
    5. const res = await fetch('/api/extract-bill', { method: 'POST', body: formData }) — do NOT set Content-Type
    6. const data: ExtractBillResult = await res.json()
    7. setResult(data); setPhase('results')
    8. Wrap in try/catch: on error setResult({ status: 'error', fields: null, message: 'Network error — check your connection.' }); setPhase('results')

    Upload UI (phase === 'upload'):
    - Header: amber eyebrow label "Bill Extractor", h1 "Upload a Utility Bill", subtitle explaining what it does
    - Drop zone card (bg-neutral-900, border-neutral-800, rounded-xl): large centered dashed border area
      - accepts drag-and-drop: onDragOver preventDefault, onDrop calls handleUpload(e.dataTransfer.files[0])
      - Hidden <input type="file" accept=".pdf,.jpg,.jpeg,.png,.heic,.heif" ref={fileInputRef} onChange={...} />
      - Click on drop zone triggers fileInputRef.current?.click()
      - Drop zone content: upload SVG icon (cloud-upload style), "Drag & drop your bill here" text, "or click to browse" subtext in neutral-500
      - Accepted formats note: "PDF, JPG, PNG, HEIC — max 4 MB" in neutral-600 text-xs
    - If uploadError: amber error panel (same style as SunPitchTab scrapeError panel)
